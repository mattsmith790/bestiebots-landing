# BestieBots.ai

Automated crypto signal execution — landing page.

## Deploy to Vercel

1. Push this repo to GitHub
2. Import into [Vercel](https://vercel.com)
3. Framework preset: **Vite** (auto-detected)
4. Click Deploy

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output goes to `dist/`.
